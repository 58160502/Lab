การใช้งาน Form

Post by Jane Dane, Sep 27, 2015.
Bootstrap Ipsum

วันนี้เราจะมาพูดถึงเรื่องการใช้งาน Form ของ Bootstrap ซึ่งก็ไม่ได้มีอะไรซับซ้อนมากมาย โดยเราแค่เรียกใช้ class “form-control” ใน textbox
combo-box หรือ textarea แค่นี้ input ต่างๆ ก็แสดงผลในรูปแบบของ bootstrap ครับ โดย input ต่างๆ ที่มีการเรียกใช้ class “form-control”
จะมีลักษณะที่โค้งมน และมีความกว้าง 100% ใน tag ที่ input นี้อยู่
เรามาดู From แบบแรกกันเลย

Inline form
เป็น from ที่มีพวก input อยู่ในแถวเดียวกัน ซึ่งหน้าตาก็จะประมาณนี้



ซึ่งเบื้องต้นก็นำ โค๊ดด้านล่างไปวางไว้ในหน้าที่มีการเรียกใช้งาน bootstrap อยู่แล้วก็สามารถใช้งานได้เลย

Horizontal form
ตัวนี้จะเป็นฟอร์มตามแนวนอน โดย input ต่างๆ ปกติ จะอยู่คนละแถวกัน จะมีอยู่แถวเดียวกันก็เพียง label เท่านั้น ซึ่ง Horizontal form นั้นมีการนำ grid system
ที่เราได้กล่าวไปแล้วใน บทความที่แล้ว เรื่อง contanier และ grid ซึ่งหน้าตาของ Horizontal form ก็จะประมาณนี้
